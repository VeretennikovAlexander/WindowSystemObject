WindowSystemObject (WSO)

 

            WindowSystemObject (WSO) - универсальный программный комплекс для обеспечения доступа к оконной подсистеме Windows на базе архитектуры COM, для создания оконных интерфейсов в программах написанных на скриптовых языках, а также в программах, написанных на других языках и в других системах программирования.

 

WindowSystemObject (WSO) provides you with an easy way to create window interfaces using scripting languages such as JScript, VBScript and other.
